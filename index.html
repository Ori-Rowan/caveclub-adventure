About the Project

The game was made by me (Orion Rowan) for the Cave Club anniversary and Robbie's birthday. The story is based on Cavetown's songs, so if some of the stuff doesn't make sense, that's because there are references to them. The project was made almost entirely in JS, and the code is pretty reusable. So, if you want to take a look and reuse it, here is the GitHub link.

All the amazing art you can see there was made by my friend and a starting artist, Ratt Kidd. You can find him as @ratt_kidd.art on Instagram, so please show him some love and support!

If you like the game and would like to support us financially, please consider donating on the Buy Me a Coffee link below. We would appreciate it a ton. Projects like these require a lot of effort and time, and every donation allows us to continue making games like these. The money will be split evenly between me and Ratt.
